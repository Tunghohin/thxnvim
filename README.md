##  How to Install

run

```sh
if command -v curl >/dev/null 2>&1; then
    bash -c "$(curl -fsSL https://raw.githubusercontent.com/Tunghohin/thxnvim/HEAD/scripts/install.sh)"
else
    bash -c "$(wget -O- https://raw.githubusercontent.com/Tunghohin/thxnvim/HEAD/scripts/install.sh)"
fi
```
